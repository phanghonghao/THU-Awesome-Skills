## General Query Templates (Shared)

### Repository Details
```
GET https://api.github.com/repos/{owner}/{repo}
GET https://api.github.com/repos/{owner}/{repo}/releases?per_page=5
GET https://api.github.com/repos/{owner}/{repo}/topics
```

### Awesome Lists by Domain
```
q=awesome+{domain}&sort=stars&order=desc&per_page=10
```

### API Rate Limit Notes
- Unauthenticated GitHub API: 60 requests/hour (per IP)
- When rate-limited, fallback to WebSearch only
- curl format: `curl -s "URL"` then pipe to python for JSON parsing
