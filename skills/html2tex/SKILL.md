# html2tex

Convert HTML (from word2html) to LaTeX + PDF via XeLaTeX.

## When to use
User wants to convert an HTML file (especially one generated by word2html) to LaTeX or PDF. Handles tables with merged cells (colspan/rowspan), embedded images, bold/italic/underline/strikethrough, and MathJax math formulas.

## Trigger
- User mentions `html2tex`
- User wants to convert HTML to LaTeX or PDF
- User says "HTML转LaTeX", "HTML转PDF"

## Command
```bash
python ~/.claude/skills/html2tex/html2tex.py <input.html> [--output-dir DIR] [--no-compile]
```

## Arguments
- `input.html` — Input HTML file path
- `--output-dir DIR` — Output directory (default: same as input)
- `--no-compile` — Only generate .tex, skip PDF compilation

## Output
- `<input>.tex` — LaTeX source (XeLaTeX + ctex)
- `<input>.pdf` — Compiled PDF (if --no-compile is not set)

## Pipeline
```
DOCX → word2html → HTML → html2tex → LaTeX + PDF
```

## Dependencies
- XeLaTeX (from TeX Live or MiKTeX)
- Python 3.6+ (stdlib only, no pip packages needed)
